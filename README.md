# 🏢 Corretaje

ERP inmobiliario diseñado para la administración integral de propiedades en arriendo.

El sistema permite gestionar de forma centralizada todos los ingresos y gastos asociados a una propiedad, automatizando gran parte del trabajo administrativo mediante el análisis de movimientos bancarios.

## 🚀 Objetivo

Reducir el tiempo dedicado a tareas administrativas y minimizar errores humanos en la gestión inmobiliaria, permitiendo que corredores y administradores se enfoquen en la atención de propietarios y arrendatarios.

## ✨ Funcionalidades

* 🏠 Administración de propiedades
* 📑 Gestión de contratos de arriendo
* 💰 Cobro de rentas
* 🔒 Control de garantías
* 💡 Registro y pago de servicios
* 🔧 Gestión de reparaciones y mantenciones
* 💵 Cálculo y control de comisiones
* 📊 Control de ingresos y egresos
* 📋 Historial financiero por propiedad
* 👥 Gestión de propietarios y arrendatarios

## 🏦 Conciliación Bancaria Inteligente

Uno de los principales objetivos del proyecto es automatizar la administración financiera mediante la importación de cartolas bancarias.

A partir de los movimientos de la cuenta bancaria, el sistema puede:

* ✅ Detectar pagos de arriendo recibidos
* ✅ Identificar pagos pendientes
* ✅ Asociar transferencias a contratos específicos
* ✅ Registrar automáticamente movimientos financieros
* ✅ Generar estados de cuenta actualizados
